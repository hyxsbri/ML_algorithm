# 1. 보험료 예측_linear_regression
## 2022.02.03 ~ 2022.02.10

간단하게 아플 가능성이 더 높아보이는 사람에게 더 높은 보험료가 책정된다고 생각할 수 있지만, 실제로 보험료 계산은 상당이 복잡하고 사람마다 각각 다르게 책정됩니다.
독립 변수 및 종속 변수가 선형 관계에 놓여있을 때 적합하고, 다른 회귀 모델과 성능을 비교하는 baseline으로 사용되는 선형 회귀 알고리즘을 데이터셋에 적용 후,
보험료 예측을 위한 작업을 진행했습니다.

sales_management_dashboard
- 교육당 월별 매출 및 객단가/ 목표 대비 매출 달성률/ 교육 취소 현황을 직관적으로 확인할 수 있는 교육 매출 대시보드 개발
(교육 매출 데이터셋 활용)

paid_media_dashboard
- 광고 유형별 연/ 월/ 일 단위 매출 도넛 차트, CPM, CTR 등 현황을 파악할 수 있는 광고 미디어 매출 대시보드 개발
(광고 미디어 매출 데이터 활용)
